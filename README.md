# Weather App (OpenWeatherMap API)
Simple Rust application to fetch and display weather data using the OpenWeatherMap API. Designed with clear separation between data models, API client, and configuration.

## Features
- Fetch current weather data from OpenWeatherMap
- Strongly-typed weather data model
- Centralized API client
- Secure API key configuration
- Clean, maintainable structure

## Requirements
- API Key from WeatherMap API
- Internet Connections

## Build
```bash
cargo build --release
```

## Usages
```bash
weather Jakarta
```
